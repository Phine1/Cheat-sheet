HTML Table

HTML tag Name Description
<table> Table The 'table' tag is declared first before creating a table.
<tr> table row Table row is used to create the rows of a table, horizontally accross the page
<th> table heading Table heading is used to give headings to table cells.
<td> table data Table data is used to enter data into a cell in a table

Note: Table headings have 'scope' attributes which indicate whether its a column or a row heading. Its uses the tags <th scope="col"> for column heading and <th scope="row"> for row headings
